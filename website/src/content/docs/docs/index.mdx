`dbt-forge` is a Python CLI that scaffolds a new dbt project and can later add marts
and sources to an existing one.

Use `init` when you want folders, profiles, starter models, and optional setup files
created for you. Use `add` when you want to extend a dbt project without rebuilding
its starting structure.

## How to use these docs

- Start with [Getting started](/docs/getting-started/) for the quickest path from install to a working dbt project.
- Use [init](/docs/cli/init/) when you are creating a new dbt project.
- Use [add](/docs/cli/add/) when you are extending an existing dbt project with a new mart or source.
- Use [Project structure](/docs/project-structure/) when you need to understand the generated layout before editing it.
