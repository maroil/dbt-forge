# Code of Conduct

## Our Standards

- Be respectful and constructive.
- Assume good intent and discuss ideas on their merits.
- Avoid harassment, discrimination, and personal attacks.

## Scope

This code of conduct applies in project spaces, issues, pull requests, and discussions related to `dbt-forge`.

## Enforcement

Project maintainers may remove comments, reject contributions, or block access for behavior that harms the project or its contributors.

## Reporting

Report unacceptable behavior through the repository maintainers via GitHub.
